📁 ARCHIVOS INCLUIDOS:
  - sistema-email-organismos.html

📋 DESCRIPCIÓN:
  Sistema completo de gestión y envío de emails a organismos beneficiarios
  del Banco de Alimentos. Incluye envío individual y grupal con interfaz
  intuitiva y responsive.

🚀 CÓMO USAR:

  1. Descargar el archivo "sistema-email-organismos.html"
  
  2. Abrir el archivo directamente en su navegador web:
     - Doble clic en el archivo
     - O clic derecho → "Abrir con" → Navegador preferido
     
  3. El sistema funcionará completamente sin necesidad de servidor

✨ CARACTERÍSTICAS:

  ✓ Envío de email individual a organismos específicos
  ✓ Envío de email grupal con selección múltiple
  ✓ Búsqueda y filtrado de organismos
  ✓ Estadísticas en tiempo real
  ✓ Interfaz responsive (móvil, tablet, desktop)
  ✓ Validación de formularios
  ✓ Notificaciones toast de confirmación
  ✓ 8 organismos de ejemplo precargados
  ✓ Diseño con colores del Banco de Alimentos

🎨 COLORES DEL SISTEMA:
  - Azul Principal: #1E73BE
  - Verde Éxito: #4CAF50
  - Naranja: #FFC107
  - Rojo Error: #DC3545
  - Gris Fondo: #F4F4F4

📱 COMPATIBILIDAD:
  ✓ Google Chrome (recomendado)
  ✓ Mozilla Firefox
  ✓ Microsoft Edge
  ✓ Safari
  ✓ Opera

💡 NOTAS IMPORTANTES:

  - Este es un sistema de DEMOSTRACIÓN
  - Los emails NO se envían realmente (solo simulación)
  - Para implementación real, integrar con:
    * SendGrid
    * AWS SES
    * Mailgun
    * O cualquier servicio SMTP

📊 DATOS DE EJEMPLO:
  El sistema incluye 8 organismos de ejemplo con:
  - Nombres realistas
  - Direcciones en Laval y Montreal
  - Emails y teléfonos
  - Responsables
  - Número de beneficiarios

🔧 PERSONALIZACIÓN:

  Para modificar los datos de los organismos:
  1. Abrir el archivo HTML en un editor de texto
  2. Buscar la sección "const organismos = ["
  3. Modificar, agregar o eliminar organismos según necesidad
