│    • Guardado automático al crear/editar/eliminar                            │
│    • Inicialización automática si no existe                                  │
│    • NO se borra nunca automáticamente                                       │
│    • Compatible con modo normal del navegador                                │
│    • NO compatible con modo incógnito (limitación del navegador)             │

│ 💡 NOTAS IMPORTANTES                                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  1. Los tipos se guardan en localStorage del navegador                       │
│  2. Cada navegador tiene su propio localStorage                              │

│  4. Usa la función exportar/importar para transferir tipos                   │

│  6. Los tipos NO se sincronizan entre dispositivos                           │
│  7. Para sincronizar, necesitarías un backend (no implementado)              │
