📐 FÓRMULA DE CÁLCULO

   Paso 1: Valor Unitario = Peso Unitario (kg) × Valor por Kg (CAD$)
   
   Paso 2: Valor Total = Stock Actual × Valor Unitario
   
   Ejemplo:
   ┌─────────────────────────────────────────────────────────────┐
   │ Producto: Arroz Blanco                                      │
   │ Stock: 100 unidades                                         │
   │ Peso Unitario: 2 kg                                         │
   │ Valor por Kg: 3.50 CAD$                                     │
   │                                                             │
   │ → Valor Unitario = 2 kg × 3.50 = 7.00 CAD$                 │
   │ → Valor Total = 100 × 7.00 = 700.00 CAD$                   │
   └─────────────────────────────────────────────────────────────┘

⚙️ REQUISITOS PARA CALCULAR VALOR

   Para que un producto tenga valor monetario necesita:

   1. ✅ Categoría con "Valor por Kg" configurado
      └─→ Ir a: Configuración → Categorías → Editar
          Establecer: "Valor por Kg" (ej: 3.50 CAD$)

   2. ✅ Producto con "Peso Unitario" definido
      └─→ Se establece automáticamente al crear producto
          Si falta: ejecutar migrarPesoUnitarioProductos()

   3. ✅ Stock Actual mayor a 0
      └─→ Se actualiza con movimientos de inventario

   ✓ Se crea un nuevo producto
   ✓ Se actualiza el stock
   ✓ Se modifica el peso unitario
   ✗ Se cambia el precio en categoría → EJECUTAR recalcularTodosLosValores()

🆘 SOLUCIÓN DE PROBLEMAS

   PROBLEMA: "Productos sin valor monetario"
   SOLUCIÓN:
   1. verificarProductosSinPeso()  ← Ver productos problemáticos
   2. migrarPesoUnitarioProductos()  ← Corregir automáticamente
   3. recalcularTodosLosValores()  ← Actualizar valores
   4. F5  ← Refrescar página

   ─────────────────────────────────────────────────────────────

   PROBLEMA: "Valores no aparecen en interfaz"
   SOLUCIÓN:
   1. recalcularTodosLosValores()  ← Forzar recalculo
   2. F5  ← Refrescar página
   3. Verificar consola (F12) por errores

   ─────────────────────────────────────────────────────────────

   PROBLEMA: "Error al calcular"
   SOLUCIÓN:
   1. F12  ← Abrir consola
   2. verificarProductosSinPeso()  ← Diagnosticar
   3. Verificar que la categoría tenga "Valor por Kg"
