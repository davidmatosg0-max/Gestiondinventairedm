## 🔍 FUNCIÓN DE BÚSQUEDA

El sistema incluye una función de búsqueda por nombre:

```typescript
import { buscarIconoPorNombre } from './data/iconosAlimentos';

// Ejemplos:
buscarIconoPorNombre('manzana')  // Retorna '🍎'
buscarIconoPorNombre('café')     // Retorna '☕'
buscarIconoPorNombre('pizza')    // Retorna '🍕'
buscarIconoPorNombre('leche')    // Retorna '🥛'
```

## 💡 CASOS DE USO

### **1. Selector de Icono en Formulario:**
```typescript
import { ICONOS_PRINCIPALES } from './data/iconosAlimentos';

// Mostrar primero los iconos principales
{ICONOS_PRINCIPALES.map(icono => (
  <button key={icono} onClick={() => setIcono(icono)}>
    {icono}
  </button>
))}
```

### **2. Selector Agrupado por Categoría:**
```typescript
import { ICONOS_POR_CATEGORIA } from './data/iconosAlimentos';

{Object.entries(ICONOS_POR_CATEGORIA).map(([categoria, iconos]) => (
  <div key={categoria}>
    <h3>{categoria}</h3>
    {iconos.map(({ emoji, nombre }) => (
      <button key={emoji} title={nombre}>
        {emoji}
      </button>
    ))}
  </div>
))}
```

### **3. Búsqueda Inteligente:**
```typescript
import { buscarIconoPorNombre } from './data/iconosAlimentos';

const icono = buscarIconoPorNombre(nombreProducto);
// Sugerencia automática basada en el nombre del producto
```
